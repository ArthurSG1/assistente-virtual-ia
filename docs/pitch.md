# Pitch do Projeto

## Problema

Muitas pessoas possuem dúvidas sobre segurança digital e acabam caindo em golpes virtuais por falta de informação.

---

## Solução

O CyberHelper AI foi desenvolvido para responder dúvidas básicas sobre segurança da informação utilizando uma base de conhecimento organizada.

O assistente fornece respostas simples, claras e evita criar informações inexistentes.

---

## Tecnologias Utilizadas

- Python
- JSON
- Markdown
- Visual Studio Code
- GitHub

---

## Diferenciais

- Interface simples.
- Base de conhecimento personalizada.
- Respostas objetivas.
- Não inventa informações.
- Fácil expansão da base de conhecimento.

---

## Conclusão

O projeto demonstra como um assistente virtual pode auxiliar usuários iniciantes na prevenção de golpes digitais utilizando uma base de conhecimento organizada e boas práticas de desenvolvimento.
